# Heartburn Checker

Step to run:

3. Access the website : localhost:3000

1. yarn test

Note:

open -n -a /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --args --user-data-dir="/tmp/chrome_dev_test" --disable-web-security
